Moodle Course Archiver - Help

Getting Started

Moodle Course Archiver helps you save course content for offline reference.

How to use

  1. Navigate to a Moodle course page
  2. Click the Moodle Course Archiver extension icon
  3. The extension will detect that you're on a Moodle course page
  4. Choose between Archive (ZIP) or Direct Download methods:
    • Archive This Course: Creates a ZIP file containing all selected content
    • Direct Download: Starts individual downloads for each file (similar to the console snippet)

Download Methods

  • Archive (ZIP): Collects all files into a single ZIP archive. Best for organizing content and reducing browser notifications.
  • Direct Download: Downloads each file individually. Similar to the console script approach and useful if you prefer files to go directly to your downloads folder.

Tip: For large courses with many files, the ZIP method may be more convenient to avoid browser download prompts for each file.
