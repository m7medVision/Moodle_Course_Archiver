Moodle Course Archiver Help

The Moodle Course Archiver extension helps you download files from Moodle learning management systems for offline access.

Note: This extension only works on Moodle course pages.

Features

Archive as ZIP

Creates a ZIP archive containing all downloadable files from the current course page. This is useful for:

  • Keeping an organized backup of course materials
  • Saving all files in one click instead of downloading them individually
  • Maintaining the course structure for offline reference

Download Files Directly

Downloads all files individually without creating a ZIP archive. This is useful when:

  • You want immediate access to individual files
  • You prefer to organize files yourself
  • You're on a system with limited space or memory

How to Use

  1. Navigate to a Moodle course page
  2. Click the extension icon in your browser toolbar
  3. Select your preferred download method:
    • Archive as ZIP: Creates a single ZIP file containing all course files
    • Download Files Directly: Downloads each file individually
  4. Wait for the download process to complete
Important: Some browsers may block multiple downloads. If prompted, allow the downloads to continue.

Troubleshooting

The extension doesn't detect my Moodle course

Make sure you're on an actual course page in Moodle. The extension looks for specific Moodle course elements.

Downloads are blocked

Your browser may block multiple downloads. Look for a notification in your browser and allow the downloads to continue.

No files are found

The extension may not be able to detect files if your Moodle site uses a heavily customized theme or structure. Some Moodle installations use different CSS selectors for file links.

Privacy

This extension only interacts with the current page you're viewing and does not send any data to external servers. All processing happens locally in your browser.
